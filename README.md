# GitHubPagesTest

# Setup
1. Import key:
```
wget "https://codecrafter912.github.io/GitHubPagesTest/pub.gpg" | sudo apt-add key -
```
2. Add repo:
```
echo "deb https://codecrafter912.github.io/GitHubPagesTest  bionic main" > /etc/apt/sources.list.d/ccTest.list
```
3. Apt update
```
apt update
```
